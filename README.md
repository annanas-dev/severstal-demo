# Severstal Demo - Иерархическая кластеризация

Streamlit-приложение для иерархической кластеризации промышленных данных.

## Запуск через Docker (рекомендуется)

### Предварительные требования
- Docker
- Docker Compose

### Быстрый старт

1. Клонируйте репозиторий и перейдите в директорию проекта

2. Запустите приложение:
```bash
docker-compose up --build
```

3. Откройте браузер и перейдите по адресу:
```
http://localhost:8502
```

### Остановка приложения
```bash
docker-compose down
```

## Запуск локально (без Docker)

### Предварительные требования
- Python 3.9+
- uv (пакетный менеджер)

### Установка зависимостей
```bash
uv sync
```

### Запуск приложения
```bash
make run
# или
uv run streamlit run main.py --server.port=8502
```

## Возможности приложения

- Загрузка данных (CSV, Excel)
- Предобработка данных
- Фильтрация по датам
- Иерархическая кластеризация
- Визуализация результатов
- Анализ важности признаков

## Технические детали

### Зависимости
Проект использует обновленные версии библиотек для совместимости с Linux/macOS:
- pandas >= 2.0.0
- numpy >= 1.24.0, < 2.0.0
- matplotlib >= 3.7.0
- scikit-learn >= 1.3.0
- scipy >= 1.11.0
- seaborn >= 0.12.0

### Кросс-платформенная совместимость
Приложение полностью совместимо с Windows, Linux и macOS. Docker образ собирается для архитектур x86_64 и ARM64.
